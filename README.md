# Future Interns - Machine Learning Internship

## Task 1: Sales Forecasting

This project analyzes sales data and predicts future sales trends using Machine Learning techniques.

## Features
- Data Cleaning
- Sales Visualization
- Monthly Sales Analysis
- Forecasting using Linear Regression
- Future Sales Prediction

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Outcome
The project predicts sales trends and helps businesses make data-driven decisions.
